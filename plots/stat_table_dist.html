Table 2. Summary of the output of linear models in the Potomac river investigation. A total of six linear models were run, with distance from the Chambersburg Wastewater Treatment Plant as the predictor variable for all of them. EPT index (Figure 7) and the five target chemistry variables (Figure 8) served as the respective response variables. For each model, the estimate value of the predictor and the intercept, 95% confidence interval values and p-values have been included. Total number of observations for all models were 6. R2 and adjusted R2 values for all models are also included, as well as the AIC~C values. The model with EPT Index as the response variable had the lowest AIC~C score. All values have been rounded to three decimal places.
